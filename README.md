## 📦 프로젝트 실행 방법

> SLAY 프로젝트를 직접 실행해보고 싶다면 아래 절차에 따라 실행해보세요.

### 🧑‍💻 환경 준비

- Node.js 18+  
- npm 9+  
- PostgreSQL 설치 및 연결 설정

### 🚀 실행 절차

```bash
# 1. 레포지토리 클론
git clone https://github.com/TeamSLAY/SLAY.git

# 2. 프로젝트 디렉토리로 이동
cd SLAY

# 3. 필요한 패키지 설치
npm install

# 4. .env 파일 설정 (예시 참고하여 환경변수 입력)

# 5. 개발 서버 실행
npm start
```
